*** Updating the "lifetimes" Library

To update the "lifetimes" Python library with the provided updated version, follow these steps:

1- Find the installation location of the "lifetimes" library by using the following command in your terminal: pip show lifetimes. This command will display details about the installed "lifetimes" library, including its location.

2- Navigate to the location of the installed "lifetimes" library.

3- Replace the entire "lifetimes" folder located at the installation path with the one provided in the "Library Update" folder.

By following these steps, you will successfully update the "lifetimes" library with the enhanced version.
